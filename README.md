# Gestion Réservations - Qt Widgets (Qt 6.9.2)

Ouvrir airline_booking.pro dans Qt Creator (sélectionnez Qt 6.9.2).
Run qmake → Rebuild Project.

Base par défaut : SQLite (fichier ./data/airline.db créé automatiquement).
Si vous voulez une connexion MySQL/XAMPP, demandez-le et j'ajoute ConnexionSQL et instructions.

Pour générer le fichier SQL initial (init.sql), demandez "génère le fichier SQL" et je vous fournirai le script SQL complet.
